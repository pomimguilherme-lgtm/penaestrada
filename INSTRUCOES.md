# Pe na Estrada - Instalacao e Execucao

## Pre-requisito
Instale o Node.js (versao 18 ou superior): https://nodejs.org

---

## 1. Instalar dependencias do Backend
```
cd backend
npm install
```

## 2. Instalar dependencias do Frontend
```
cd frontend
npm install
```

## 3. Rodar o projeto

### Terminal 1 - Backend
```
cd backend
npm run dev
```
O servidor inicia em: http://localhost:3001

### Terminal 2 - Frontend
```
cd frontend
npm run dev
```
O app abre em: http://localhost:3000
